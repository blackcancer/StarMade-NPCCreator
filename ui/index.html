📋 Dialog Structure
  NPC Script     — root container (one per script)
  Greeting       — opening text shown to player
  Choice         — a button option the player can click
  Go Back        — adds a "back" button to previous menu

🧮 Variables
  Variable       — floating global Lua variable / constant

👥 Crew Actions
  Hire Crew      — player pays to hire this NPC
  Dismiss Crew   — remove NPC from player's team

🛒 Items to Sell
  Sell Item      — give a weapon/tool for credits
  Give Block     — give any block type by ID

⚡ World Actions
  Activate Block — toggle a block on/off
  Move NPC To    — NPC walks to block coordinates

💾 SQLite State
  SQLite Table   — define a persistent data table
  Get Value      — read a value for this player
  Set Value      — write a value for this player
  Increment      — add N to a counter

🔧 Advanced
  Custom Hook    — write raw Lua code for a hook

Placeholders in text:
  {name}    → NPC's name
  {partner} → NPC's affinity
  {faction} → NPC's faction name
  {owner}   → NPC's owner name
  {self}    → Player's name
